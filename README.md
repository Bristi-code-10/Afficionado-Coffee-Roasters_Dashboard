# ☕ Coffee Shop Sales Dashboard

An interactive data analytics dashboard built using Streamlit to analyze coffee shop sales data and generate business insights.

---

## 📊 Project Overview
This project focuses on analyzing transaction data from a coffee shop to understand:
- Sales trends over time  
- Peak business hours  
- Popular products  
- Revenue patterns  

The dashboard provides a simple and interactive way to explore the data.

---

## 🚀 Features
- 📈 Sales trend analysis (daily/monthly)  
- ⏰ Peak hours identification  
- 🛒 Product performance insights  
- 📊 Interactive charts and filters  
- 💡 User-friendly dashboard interface  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib / Plotly  
- Streamlit  

---

## 📂 Dataset
- Contains transaction-level data  
- Includes fields like:
  - Date / Time  
  - Product  
  - Quantity  
  - Revenue
